# Password Generator

A simple and interactive **Password Generator** built using HTML, CSS, and JavaScript. This application helps users generate strong and secure passwords with customizable options.

##  Features

*  Adjustable password length (6–24 characters)
*  Include uppercase letters
*  Include lowercase letters
*  Include numbers
*  Include symbols
*  Copy password to clipboard
*  Password strength indicator (Weak / Medium / Strong)
*  Clean and responsive UI

##  Technologies Used

* HTML
* CSS
* JavaScript
* Font Awesome (for icons)
##  How It Works

* Select the desired password length using the slider
* Choose the options (uppercase, lowercase, numbers, symbols)
* Click on **Generate Password**
* Copy the generated password using the copy icon
* Check password strength using the indicator
